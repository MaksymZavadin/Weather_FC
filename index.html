Notes & help
  • Historical data: 1940 → near‑present, with a few‑day delay depending on dataset. For the very latest days, use Forecast API with past_days.
  • Inputs accept ISO dates (YYYY‑MM‑DD). For hourly data, times start at 00:00 local time if timezone is set.
  • You can switch between hourly and daily outputs via the Granularity selector. Make sure your variable names match the selected granularity.
  • This tool uses the Open‑Meteo Geocoding API to resolve city names to coordinates and timezone.
